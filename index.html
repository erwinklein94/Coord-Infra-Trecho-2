Integração

Fonte única da verdade: planilha PDM

O site foi preparado para ler a planilha online publicada em CSV. Enquanto as URLs estiverem vazias, ele usa os dados de exemplo extraídos da planilha anexada.

URLs das abas publicadas em CSV

Depois de publicar a planilha PDM na web, cole aqui a URL CSV de cada aba. O navegador salva essa configuração localmente. Para deixar centralizado para todos, edite o arquivo data/source-config.json.

Fluxo recomendado

Fiscais enviam dados padronizados → analista atualiza a planilha PDM principal → o site lê a aba online e recalcula automaticamente os cards.
